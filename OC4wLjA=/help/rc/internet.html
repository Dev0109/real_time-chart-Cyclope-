Internet si Aplicații Web

Informaţiile incluse în această pagină:
  • Structură arborescentă conţinând Conturile de login, Utilizatorii şi Calculatoarele analizate organizate pe file separate.

    • Domenii web
  • Grafic cu bare ilustrând primele 5 cele mai vizitate domenii web, grupate după Categoria care le-a fost atribuită.
  • Grafic radial ilustrând utilizarea tuturor domeniilor web vizitate, grupate după statutul de operativitate atribuit (operativ, neoperativ sau neutru).
  • Tabel de date conţinând următoarele informaţii afişate pe coloane:
    • Domeniu web - Numele domeniului web înregistrat.
    • Categorie - O selecţie de tip dropdown unde utilizatorul poate atribui unei aplicaţii una dintre categoriile definite. Valoarea implicită este Fără categorie.
    • Timp - Timpul de utilizare pentru domeniul înregistrat.
    • % din Timpul activ - Cât din Timpul activ reprezintă timpul înregistrat pentru fiecare domeniu web.

    • Link-uri
  • Tabel de date conţinând următoarele informaţii afişate pe coloane:
    • Link pagină web - URL-ul complet al paginii web înregistrate.
    • Timp - Timpul de utilizare pentru fiecare pagină web.
    • % din Timpul activ - Cât din Timpul activ reprezintă timpul înregistrat pentru fiecare pagină web.

    • Titluri pagini
  • Tabel de date conţinând următoarele informaţii afişate pe coloane:
    • Fereastra - Titlul paginii web înregistrate.
    • Timp - Timpul de utilizare pentru fiecare pagină web.
    • % din Timpul activ - Cât din Timpul activ reprezintă timpul înregistrat pentru fiecare pagină web.

Selectorul datei calendaristice oferă câteva intervale de timp predefinite (Azi, Ieri, Săptămâna aceasta, Săptămâna trecută, Luna aceasta şi Luna trecută) la fel ca şi posibilitatea de a selecta o Dată specifică sau un Interval de date.

Un filtru sub formă de listă dropdown care permite utilizatorului să filtreze informaţia din tabel după oricare din aplicaţiile înregistrate de pe staţiile de lucru analizate şi care au tipul setat pe 'Browser'.

Administratorul are permisiunea de a vizualiza toate statisticile. Managerul poate vizualiza doar statisticile care aparţin utilizatorilor care fac parte din departamentele pe care le gestionează el. Utilizatorul are permisiunea de a vizualiza doar statisticile unui singur utilizator (în mod normal, pe cele ale sale).

Această pagină poate fi exportată în formate CSV, XLS şi PDF făcând click pe iconiţa corespunzătoare localizată în dreapta paginii, sub titlul paginii şi în continuarea filtrelor pentru timp.

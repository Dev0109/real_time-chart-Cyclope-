Client

  • 1. Path - The file path to the Cyclope Client setup on your server.
  • 2. Cyclope client - The executable file you need to run on the stations you want to monitor.
  • 3. Server Name - When installing the client, make sure you fill this box with the name or IP of your server.
  • 4. Test Server Connection - Check the connection to the server by pressing this button.
  • 5. Save Settings - Save your options and complete the client installation process.

Software Inventory

Information sections included in this page:
  • Tree View containing monitored Sessions, Employees and Computers, organized on separate tabs.
    • Computer - Employee - IP
  • Basic information about the employee:
    • Computer name
    • Windows login account name
    • The IP address used in the local network
    • System Information
  • Data Table containing the following information displayed on columns:
    • OS - Operating System
    • CPU - Microprocessor details
    • Memory - Available RAM
    • Storage - Installed storage units
    • Monitor - Display monitor details
    • Installed Applications
  • Data Table containing the list of all software applications installed. Details for each entry:
    • Application Name - Name of the installed application
    • Publisher - The name of the publisher of the applicaiton
    • Install date - The date when the application was installed
This report is available only per Employee.

This page can be exported in CSV, XLS and PDF formats by clicking the corresponding icon located right under the page title and near the time filters.

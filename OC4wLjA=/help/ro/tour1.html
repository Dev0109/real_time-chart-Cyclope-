Overview

  • 1. TreeView - Always showing you the selected monitored session (User+Computer) or selected department. Reffer to the Users and Computers tabs for individual information on the monitored users and computers.
  • 2. Datetime picker - Select the date here. Choose from predefined time frames, specific dates or date range.
  • 3. Charts and data area - See here the returned data organized in charts and data tables.
  • 4. Export icons - Export the displayed report to CSV or PDF format.
  • 5. Menu - Choose the desired report you would like to see from this menu.
